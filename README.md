# College of Education, Kura Website (GitHub Auto Deploy)

This version is fully configured for GitHub Pages auto-deployment.

## Deployment URL
https://ayassinkar-max.github.io/coe-kura/

## Steps
1. git init
2. git add .
3. git commit -m "Initial upload"
4. git branch -M main
5. git remote add origin https://github.com/ayassinkar-max/coe-kura.git
6. git push -u origin main

GitHub Actions will automatically deploy your site within minutes.
